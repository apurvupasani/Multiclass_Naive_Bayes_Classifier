######################################

nblearn.py

Executed as : python nblearn.py <training_file_name_with_path> <model_file_name>

This python class takes training file(with relative path) and the model file name to be generated(again with full path) and generates the model file for both SPAM and Sentiment Data Set


######################################

nbclassify.py

Executed as : python nbclassify.py <model_file_name> <test_file_name>

This python class classifies the test file(with relative path) according to model file. It prints the classification label as output on STDOUT


######################################


spam.nb

Model file of the spam training set


########################################

sentiment.nb

Model file of sentiment training set

########################################

spam.out

Output of the nbclassify.py for spam test set

###########################################

sentiment.out

Output of the nbclassify.py for sentiment test set
